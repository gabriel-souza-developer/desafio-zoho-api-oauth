FLUXO DO PROJETO: 

1. Acessa http://localhost:8000/index.php.
2. Clica no botão “Login Zoho”.
3. Faz login → Zoho redireciona para http://localhost:8000/callback.php?code=xxxx.
4. O PHP troca esse code pelo access_token.
5. Exibe o token na tela (só pra teste).
