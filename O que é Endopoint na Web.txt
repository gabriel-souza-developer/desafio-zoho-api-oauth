 O que é um endpoint

Um endpoint é basicamente um “endereço de internet” que você chama para interagir com uma API.

Pense assim: a API é como um restaurante.

O endpoint é a porta ou balcão específico onde você pede algo. Cada endpoint tem uma função específica.

No nosso caso:

POST https://accounts.zoho.com/oauth/v2/token


Esse endpoint do Zoho é o balcão que recebe o code e devolve o access_token.

Você envia dados (client_id, client_secret, code, etc.) e recebe uma resposta em JSON.
